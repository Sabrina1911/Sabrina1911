**Featured Project â€” CarbonLensAI**

CarbonLensAI is an AI-driven prompt optimization and efficiency analysis system inspired by the Green AI philosophy.
The system is implemented using a modular Python pipeline with a Streamlit-based UI for interactive prompt optimization and evaluation.

*What it does:*

* Safely optimizes LLM instruction prompts using a learned T5 rewriter
* Preserves protected payloads (e.g., articles, transcripts) byte-for-byte
* Applies semantic safety gates (embedding cosine similarity)
* Compares rule-based baselines vs learned optimization
* Reports token reduction, efficiency gains, and trade-offs

*Why it matters:*

* Demonstrates governed AI design (not blind optimization)
* Connects model efficiency to energy and carbon-aware thinking
* Designed for auditability, reproducibility, and academic evaluation

This project reflects my interest in building responsible, efficiency-aware AI systems that balance performance, safety, and real-world constraints.
